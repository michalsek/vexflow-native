# vexflow-native

React native backend for rendering music notation using @shopify/react-native-skiaact-native-skia

## Installation


```sh
npm install vexflow-native
```


## Usage


```js
import { multiply } from 'vexflow-native';

// ...

const result = await multiply(3, 7);
```
